# FlowCraft Web

Next.js (App Router) scaffold for FlowCraft UI. Uses Tailwind CSS, Zustand, and React Flow with design tokens from `../docs/figma-spec.md`.

## Getting Started

```bash
pnpm install
pnpm dev
```

Environment:
- `NEXT_PUBLIC_API_BASE_URL` (see `.env.example`)

Structure follows feature folders with presentational components separated from hooks/stores/services.
